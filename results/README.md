# Results

Benchmarking outputs are saved here by default.

Expected files include:

- `{model}_{dataset}_results.txt`
- `{model}_{dataset}_confusion_matrix.csv`
- `{model}_{dataset}_confusion_matrix.png`
- `{model}_{dataset}_label_mapping.json`
- `{model}_{dataset}_config.json`
- `benchmark_summary.csv`

Generated result files are ignored by Git; this README is kept so the folder is visible.
