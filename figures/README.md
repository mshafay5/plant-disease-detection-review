# Figures

This folder contains the three featured figures used on the repository landing page.

- `figure_2_taxonomy.png`: taxonomy of plant disease detection literature.
- `figure_7_rgb_timeline.png`: chronological overview of RGB-based plant disease detection methods.
- `figure_10_hsi_timeline.png`: chronological overview of HSI-based plant disease detection methods.

The figures are copied from the paper source without altering their scientific content.
