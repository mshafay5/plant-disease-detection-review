# Tables

This folder contains machine-readable resources derived from the paper.

- `datasets.csv`: derived from Table 2, summarizing plant disease datasets, environments, sizes, annotations, characteristics, challenges, citation keys, and updated dataset links.
- `benchmark_results_table6.csv`: derived from the classification performance comparison table, reporting Accuracy/F1-score values across benchmark datasets.

Dataset links are maintained in [`../docs/dataset_links.md`](../docs/dataset_links.md) and mirrored in the `Link` column of `datasets.csv`.
