# Contribution Guide

Contributions are welcome when they improve the usefulness and accuracy of this companion repository.

Useful contributions include:

- adding verified dataset links for entries currently marked `TODO`;
- suggesting missing recent plant disease detection papers;
- correcting citation metadata;
- reporting issues with the benchmarking script;
- adding reproducibility notes for specific datasets.

Please include a source for any correction, preferably a DOI, official dataset page, publisher page, or repository maintained by the dataset authors. Do not upload datasets, trained checkpoints, or copyrighted material unless redistribution is explicitly permitted.
